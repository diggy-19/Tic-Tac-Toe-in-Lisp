Tic-Tac-Toe-in-Lisp
===================

A game of Tic-Tac-Toe writen in Common Lisp for an AI course.

Logic
-----

You can find the logic on [Google Drive][1].

How to Run
----------

 *  ### On Windows
    *Run* `run.bat clisp`

 *  ### On Mac OSX / OSX / Linux
    *Run* `run.sh clisp`
